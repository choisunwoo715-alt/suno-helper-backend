## 🎉 v11.0 주요 업데이트

### 🌍 **글로벌 히트 팩 6종 추가!**

1. **Hip-Hop Boom Bap** (90s 골든 에이지)
   - AABB/ABAB 라임 스킴
   - 7-12음절 유동적 구조
   - 명확한 발음, 자신감 있는 전달

2. **Melodic Trap** (감성 래핑)
   - Triplet flow (3연음)
   - Auto-Tune 멜로딕 랩
   - 노래+랩 혼합

3. **Funk Pop** (브루노 마스 스타일)
   - Call-and-Response 구조
   - Syncopation (엇박)
   - 그루비한 파티 바이브

4. **Reggaeton** (라틴 댄스)
   - Dembow rhythm (3+3+2)
   - Hook 반복 최소 3회
   - 래핑+노래 혼합

5. **Future Bass** (EDM 팝)
   - 극도로 짧은 구절 (4-8음절)
   - Vocal chops 최적화
   - Drop 보컬 최소

6. **Indie Pop (Bedroom Pop)**
   - 진솔하고 취약한 가사
   - Lo-fi 감성
   - 자유로운 구조 (3-10음절)
