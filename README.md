## ⚙️ 장르별 프롬프트 차이점

### K-Pop 댄스
- **음절 제한 없음!** (리듬 우선)
- 후크 2-3음절 반복 (나나나, 예예예)
- Final Chorus **12줄까지** 허용
- 랩 아님, 댄스 리듬!

### 로파이 힙합
- **가사 최소화!** (인스트루멘탈 중심)
- 허밍, 속삭임 위주
- Final Chorus **4줄만**
- "hmm...", "yeah..." 같은 최소 보컬

### 시티팝
- **4-6음절** (발라드보다 여유)
- 도시적 낭만, 재즈 감성
- Final Chorus **10줄**
- 직설적 슬픔 X, 세련된 분위기

### 발라드 계열 (6종)
- **3-5음절 엄수**
- Final Chorus **8-9줄**
- 트로트는 꺾기 강조
- 록은 Final만 고음
