Suno Helper Pro

이용약관

최종 수정일: 2025년 1월 1일

1. 약관 동의

Suno Helper Pro (이하 "서비스")를 사용함으로써 귀하는 본 이용약관에 동의하는 것으로 간주됩니다. 동의하지 않을 경우 서비스를 사용할 수 없습니다.

2. 서비스 설명

Suno Helper Pro는 다음 서비스를 제공합니다:

  • AI 기반 한국 발라드 및 K-Pop 가사 자동 생성
  • 17개 장르 지원
  • 8대 원칙 기반 품질 보장
  • Gemini API 연동

3. 사용자 책임

사용자는 다음 사항에 동의합니다:

  • API 키 관리: 자신의 Gemini API 키를 안전하게 관리할 책임
  • 적법한 사용: 생성된 가사를 합법적 목적으로만 사용
  • 저작권 존중: 타인의 저작권을 침해하지 않음
  • 서비스 남용 금지: 과도한 요청, 자동화 스크립트 사용 금지

4. 지적재산권

생성된 가사:

  • 사용자가 생성한 가사의 저작권은 사용자에게 귀속됩니다.
  • Suno Helper Pro는 생성된 가사에 대한 권리를 주장하지 않습니다.
  • 단, 서비스 개선 목적으로 익명화된 통계 데이터는 사용될 수 있습니다.

서비스 코드 및 알고리즘:

  • Suno Helper Pro의 코드, 알고리즘, 디자인은 저작권법으로 보호됩니다.
  • 무단 복제, 배포, 수정을 금지합니다.

5. 서비스 제한 및 중단

Suno Helper Pro는 다음 경우 서비스를 제한하거나 중단할 수 있습니다:

  • 시스템 유지보수 및 업그레이드
  • 법적 요구 또는 정부 명령
  • 약관 위반 사용자
  • 기술적 문제 발생 시

서비스 중단으로 인한 손해에 대해 Suno Helper Pro는 책임지지 않습니다.

6. 레이트 리밋

무료 사용자는 시간당 20회의 가사 생성 제한이 있습니다. 과도한 사용 시 일시적으로 차단될 수 있습니다.

7. 면책 조항

Suno Helper Pro는 다음 사항에 대해 책임지지 않습니다:

  • 생성 품질: AI가 생성한 가사의 품질, 정확성, 적합성
  • Gemini API 문제: Google Gemini API의 장애, 오류, 변경
  • 저작권 침해: 사용자가 생성한 가사로 인한 저작권 분쟁
  • 데이터 손실: 브라우저 오류 등으로 인한 가사 손실

8. 환불 정책

현재 Suno Helper Pro는 무료 서비스입니다. 향후 유료 플랜 도입 시 별도 환불 정책이 공지됩니다.

9. 약관 변경

이 약관은 사전 고지 없이 변경될 수 있습니다. 중요한 변경 사항은 홈페이지를 통해 공지됩니다.

본 약관은 대한민국 법률에 따라 해석되며, 분쟁 발생 시 서울중앙지방법원을 제1심 관할 법원으로 합니다.
