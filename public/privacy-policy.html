Suno Helper Pro

개인정보처리방침

최종 수정일: 2025년 1월 1일

1. 수집하는 정보

Suno Helper Pro는 다음과 같은 정보를 수집합니다:

  • 사용자 제공 정보: Gemini API 키 (클라이언트 측에만 저장)
  • 자동 수집 정보: IP 주소, 브라우저 정보, 사용 통계
  • 생성 데이터: 가사 생성 요청 내역 (서버에 저장되지 않음)

2. 정보 사용 목적

  • 서비스 제공 및 개선
  • 사용자 경험 최적화
  • 기술적 문제 해결
  • 서비스 품질 모니터링

3. 정보 보관 및 보호

중요: Suno Helper Pro는 사용자의 개인정보를 서버에 저장하지 않습니다.

  • API 키: 브라우저 로컬 스토리지에만 저장 (서버 전송 없음)
  • 생성된 가사: 일시적으로 전송되지만 저장되지 않음
  • IP 주소: 레이트 리밋 목적으로만 사용 (24시간 후 자동 삭제)

4. 제3자 공유

Suno Helper Pro는 사용자 정보를 제3자와 공유하지 않습니다. 단, 다음의 경우 예외입니다:

  • Google Gemini API: 가사 생성을 위해 사용자의 키워드가 전송됨
  • Vercel (호스팅): 서버 로그 (IP, 사용 통계)
  • 법적 요구: 법원 명령 등 법적 의무가 있는 경우

5. 쿠키 및 추적 기술

Suno Helper Pro는 최소한의 쿠키만 사용합니다:

  • 필수 쿠키: 세션 관리, 레이트 리밋 (삭제 불가)
  • 선택 쿠키: 사용하지 않음

6. 사용자 권리 (GDPR 준수)

유럽연합(EU) 사용자는 다음 권리를 가집니다:

  • 접근권: 수집된 정보 확인
  • 삭제권: 저장된 정보 삭제 요청 (브라우저에서 직접 가능)
  • 정정권: 잘못된 정보 수정
  • 이동권: 데이터 다운로드 (가사 복사)

권리 행사 문의: support@suno-helper.com

7. 어린이 개인정보

Suno Helper Pro는 만 13세 미만 어린이의 개인정보를 의도적으로 수집하지 않습니다.

8. 정책 변경

이 개인정보처리방침은 수시로 변경될 수 있습니다. 중요한 변경 사항은 홈페이지를 통해 공지합니다.

Privacy Policy

Last Updated: January 1, 2025

1. Information We Collect

Suno Helper Pro collects the following information:

  • User-Provided: Gemini API key (stored client-side only)
  • Automatically Collected: IP address, browser info, usage statistics
  • Generated Data: Lyrics generation requests (not stored on server)

2. How We Use Your Information

  • Provide and improve services
  • Optimize user experience
  • Resolve technical issues
  • Monitor service quality

3. Data Storage and Protection

Important: Suno Helper Pro does NOT store personal information on our servers.

  • API Keys: Stored in browser local storage only (never transmitted to server)
  • Generated Lyrics: Temporarily transmitted but not stored
  • IP Addresses: Used for rate limiting only (auto-deleted after 24 hours)

4. Third-Party Sharing

Suno Helper Pro does not share user information with third parties, except:

  • Google Gemini API: Keywords transmitted for lyrics generation
  • Vercel (Hosting): Server logs (IP, usage stats)
  • Legal Requirements: Court orders or legal obligations

5. Cookies and Tracking

Suno Helper Pro uses minimal cookies:

  • Essential Cookies: Session management, rate limiting (cannot be disabled)
  • Optional Cookies: Not used

6. Your Rights (GDPR Compliance)

EU users have the following rights:

  • Right to Access: View collected information
  • Right to Deletion: Request data deletion (can be done directly in browser)
  • Right to Rectification: Correct inaccurate information
  • Right to Portability: Download data (copy lyrics)

Contact: support@suno-helper.com

7. Children's Privacy

Suno Helper Pro does not knowingly collect information from children under 13.

8. Policy Changes

This Privacy Policy may be updated periodically. Significant changes will be announced on our homepage.
